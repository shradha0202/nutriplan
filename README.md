NutriPlan — Nutrition Planning for Real Indian Diets

NutriPlan is a multi-page, interactive nutrition planning frontend built for realistic Indian dietary habits.
The project includes customizable meal recommendations, a progress tracking dashboard, and a C-logic showcase demonstrating how data structures could power backend algorithms.

This repository contains only the frontend implementation.

Features
Goal-Based Meal Planning
Users can choose:
Weight Loss
Muscle Gain
Maintenance
PCOD-Friendly
General Healthy Eating
Filter System
Selections include:
Diet preference (veg, non-veg, egg)
Budget range
Meal timing (morning, afternoon, evening)
Meal Suggestions
Dynamic meal cards
Aesthetic image-based layout
Filters stored using LocalStorage
Progress Tracker
Daily calorie, protein, and carb tracking
Progress bars with dynamic updates
Weekly calorie chart using Chart.js
Meal logging with timestamps
Automatic streak tracking
C Logic Showcase
A separate section demonstrating how data structures can support recommendation algorithms.
Includes complete C implementations for:
Binary Search Tree
Priority Queue
Stack
Linked List
Graph (Adjacency List)
